# Python基于PyQt5的图书管理系统 python684

## 项目概述

本项目是一个基于Python和PyQt5的图书管理系统，旨在帮助用户进行图书和读者信息的维护工作。系统提供图书与读者信息的增删改查功能，并支持从文件导入信息，适合初学者学习和实践。

## 技术栈

- Python
- PyQt5

## 系统角色

本系统主要分为以下角色：

- 系统管理员：负责整个图书管理系统的维护和读者信息的管理。
- 读者：可查询图书信息，但无法修改。

## 运行环境

- Python 3
- PyQt5

## 功能模块

- 图书信息管理：支持图书的增加、删除、修改和查询操作。
- 读者信息管理：支持读者信息的增加、删除、修改和查询操作。
- 信息导入：支持从文件中导入图书和读者信息。


## 核心亮点

- 界面友好：基于PyQt5的图形界面设计，操作直观简便。
- 易于学习：代码结构清晰，适合新手学习和理解Python和PyQt5编程。
